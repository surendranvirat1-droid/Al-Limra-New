# My Product Shop

A responsive product selling website built with HTML, CSS, and JavaScript. This project includes:

- Product CRUD operations (create, edit, delete)
- Click item to add directly to cart
- Cart billing with print and clear cart
- Payment section with QR code and GPay support
- Sales tracking: daily, weekly, monthly, yearly
- Date range filter for sales history
- Mobile, tablet, and desktop responsive design

## Files

- `index.html` — main website layout
- `styles.css` — responsive theme and styling
- `script.js` — product management, cart, billing, and sales logic

## Run locally

Open `index.html` in your browser.

## Deploy to Netlify or Vercel

### Netlify
1. Sign in to Netlify.
2. Create a new site from Git.
3. Connect your repository and choose the root folder.
4. Set build command to none and publish directory to `/`.

### Vercel
1. Sign in to Vercel.
2. Create a new project and import your repository.
3. No build settings are required for plain HTML/CSS/JS.
4. Deploy and share the generated URL.

## Notes

- Product data and sales records are stored in browser local storage.
- The app is designed for a free static hosting environment.
